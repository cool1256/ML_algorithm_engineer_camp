# 线性回归方法简介

# 形式化定义
+ 假设函数（hypotheses function）  
$$h_{\theta}(x)=\sum_{i=0}^{n} \theta_{i} x_{i}=\theta^{T} x$$  其中  x_{0}=1 
+ 损失函数（loss function）  
$$L(\theta)=\left(h_{\theta}(x)-y\right)^{2}$$
+ 代价函数（cost function）  
$$J(\theta)=\frac{1}{2 \mathrm{m}} \sum_{i=1}^{m}\left(h_{\theta}\left(x^{(i)}\right)-y^{(i)}\right)^{2}$$
